# Community Themes

Komari supports custom themes. Community themes can change the dashboard look and add theme-specific settings.

## Before Installing a Theme

- Download themes from trusted sources.
- Inspect raw HTML or JavaScript if the theme includes custom admin pages.
- Confirm that the theme supports your Komari server version.
- Back up data before switching major theme versions.

## Theme Development

See [Theme Development](/en/dev/theme) if you want to build or adapt a theme.
