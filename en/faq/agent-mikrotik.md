# Mikrotik Agent

Mikrotik environments are usually more constrained than a standard Linux server. Use this guide as an entry point for connecting compatible agent or script-based reporting workflows.

## Checklist

- Confirm the device model and architecture.
- Confirm which execution environment is available.
- Decide whether you will run a native agent, a helper host, or HTTP-based reporting.
- Keep reporting intervals conservative.

For detailed Chinese notes, see [Mikrotik agent guide](/faq/agent-mikrotik).
